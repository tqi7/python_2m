اهلا بك عزيز الطالب في مشروع المهارات الرقمية (عملي)
في هذا المشروع سوف نقوم بتشغيل كود بلغة البرمجة بايثون

```
for i in range(6):
    for j in range(i):
        print ('* ', end="")
    print()

for i in range(6,0,-1):
    for j in range(i):
        print('* ', end="")
    print()
```
